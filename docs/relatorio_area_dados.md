# Repositório de Consultas SQL

<!-- ## 🎯 O Porquê -->

Esse repositório é a nossa fonte da verdade. O objetivo é simples: **acabar com query perdida em e-mail ou área de trabalho e garantir que todo mundo use a mesma regra.**

Aqui esta tudo centralizado para:

*   Facilitar a vida de quem tá chegando agora.
*   Evitar retrabalho. Se já existe, usa e melhora.

## 📂 Estrutura de Navegação

O repositório está organizado por domínios de negócio. Abaixo, um guia para encontrar o que você precisa:

| Pasta | Área de Negócio / Descrição |
| :--- | :--- |
| `BI/` | Consultas otimizadas para alimentação de Dashboards e relatórios gerenciais (ex: Inadimplência, Produtos). |
| `CONTABIL/` | Fechamentos contábeis, IOF, PDD, Direito Creditório e conciliações financeiras. |
| `FINANCEIRO/` | Fluxo de recebimentos diários, pagamentos Totvs e conciliação. |
| `JURIDICO/` | Levantamentos para suporte legal e análise de processos. |
| `LIABILITY/` | Consultas específicas para a area de Liability. |
| `OPERACIONAL/` | Emissões, Viagem, Vida e RCO. |
| `PICPAY/` | Consultas dedicadas à parceria e produtos PicPay. |
| `RESSEGURO/` |Contratos de resseguro (ex: Munich). |
| `SINISTRO/` | Acompanhamento de de sinistros. |
| `VIAGEM/` | Relatórios específicos para o produto Viagem. |
| `00.BACKUP/` | *Área técnica*: Histórico, ajustes pontuais e scripts legados. |
| `01.DIVERSOS/` | *Sandbox*: Consultas diversas, desde Premrec a Resseguro Cyber. |
